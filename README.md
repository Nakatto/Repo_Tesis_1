# ESP32 Hello World Project

This project demonstrates a simple "Hello, World!" application for the ESP32 using the Arduino framework.

## Project Structure

```
esp32-hello-world
├── src
│   └── main.cpp        # Main entry point for the application
├── platformio.ini      # PlatformIO configuration file
└── README.md           # Project documentation
```

## Requirements

- ESP32 development board
- PlatformIO IDE or compatible environment

## Setup Instructions

1. **Install PlatformIO**: If you haven't already, install PlatformIO IDE or the PlatformIO extension for your preferred code editor.

2. **Clone the Repository**: Clone this repository to your local machine.

   ```
   git clone <repository-url>
   ```

3. **Open the Project**: Open the project folder in PlatformIO.

4. **Select the Board**: Ensure that the ESP32 board is selected in the PlatformIO IDE.

5. **Upload the Code**: Connect your ESP32 board to your computer and upload the code by clicking the upload button in the PlatformIO interface.

## Running the Application

Once the code is uploaded, open the Serial Monitor in PlatformIO to see the output. You should see "Hello, World!" printed to the monitor.
